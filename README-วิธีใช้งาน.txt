วิธีใช้งานเว็บ Anz & Melody
============================

ไฟล์ทั้งหมด:
- index.html     -> หน้าแรก (หน้า start)
- year.html      -> หน้าความทรงจำของ "ทุกปี" ในไฟล์เดียว แยกด้วย ?year=1, ?year=2, ?year=3 ...
- year-data.js   -> เนื้อหาของแต่ละปี (ข้อความ, wishlist ฯลฯ) แก้ตรงนี้เวลาจะเพิ่ม/แก้ปี
- style.css      -> ธีมกลางของทั้งเว็บ
- script.js      -> ตัวนับวัน, กล่องข้อความ, ระบบดูรูปแบบเต็มจอ ฯลฯ

วิธีทำงานของ year.html:
  เปิด year.html?year=1   -> แสดงเนื้อหาปีที่ 1
  เปิด year.html?year=2   -> แสดงเนื้อหาปีที่ 2
  เปิด year.html?year=3   -> ยังไม่มีเนื้อหาจริงก็ไม่พังนะ จะขึ้นเทมเพลตว่างๆ
                             ให้เข้าไปแก้ที่ year-data.js อัตโนมัติ
  ด้านบนหน้ามีแถบ "ปี 1 / ปี 2 / + ใหม่" ให้กดสลับปีได้เร็วๆ
  ด้านล่างมีปุ่ม "ปีถัดไป →" กดไปปีถัดไปได้เรื่อยๆ ไม่จำกัด

ต้องทำก่อนใช้งาน:
1) ตั้งวันเริ่มนับใน script.js บรรทัด:
       const START_DATE = "2025-07-18";
   แก้เป็นวันที่คุณสองคนเริ่มคบกันจริงๆ (รูปแบบ YYYY-MM-DD)

2) ใส่รูปภาพ (โฟลเดอร์คำนวณอัตโนมัติทีละ 4 โฟลเดอร์ต่อปี):
   - ปีที่ 1 -> โฟลเดอร์ 1, 2, 3, 4   (ไฟล์ 0.jpg–3.jpg ต่อโฟลเดอร์ ตามที่มีอยู่แล้ว)
   - ปีที่ 2 -> โฟลเดอร์ 5, 6, 7, 8
   - ปีที่ 3 -> โฟลเดอร์ 9, 10, 11, 12  (รูปแบบเดิมไปเรื่อยๆ ไม่ต้องแก้โค้ด)
   - แตะที่รูปในเว็บจะเปิดดูแบบเต็มจอ เลื่อนดูรูปถัดไปในโฟลเดอร์เดียวกันได้

3) เพิ่มเนื้อหาปีใหม่ในอนาคต:
   เปิด year-data.js แล้วเพิ่ม key ใหม่ในตัวแปร YEARS เช่น
       3: {
         title: "ครบรอบ 3 ปี",
         heroEmoji: "💜✨",
         memoryParagraphs: ["...", "..."],
         loveMessage: `"..."`,
         wishList: ["...", "..."],
       },
   ถ้ายังไม่เพิ่ม ระบบจะโชว์เทมเพลตว่างให้อัตโนมัติ ไม่ต้องกลัวเว็บพัง

ฟีเจอร์ในเว็บ:
- หน้า start แยกต่างหาก แล้วค่อยกดเข้าไปแต่ละปีผ่าน year.html?year=N
- แถบสลับปีด่วนด้านบน (ปี 1 / ปี 2 / + ใหม่)
- ตัวนับ "วันที่รักกันมา" แบบเรียลไทม์ พร้อมดวงจันทร์ที่ค่อยๆ เปลี่ยนเงาไปเรื่อยๆ
- แตะที่รูป 🖤 คุโรมิ เพื่อสุ่มข้อความน่ารักๆ
- กล่อง "ฝากข้อความถึงกัน" พิมพ์แล้วบันทึกไว้ในเบราว์เซอร์ (ถาวรทุกครั้งที่เปิดเว็บ)
- ดีไซน์จัดใหม่ทั้งหมดให้เหมาะกับจอมือถือ Android

วิธีอัปโหลด:
อัปโหลดไฟล์ทั้งหมด (รวมโฟลเดอร์รูปทั้งหมด) ขึ้นไปที่โฮสต์เดิมของคุณ ให้ทุกไฟล์อยู่ตำแหน่งเดียวกัน (ระดับเดียวกับ index.html) เหมือนเดิม
